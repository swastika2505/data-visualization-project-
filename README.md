# Data Cleaning & Visualization Project

##  Project Title

Data Cleaning and Visualization of FIFA World Cup 2026 Player Performance Dataset

##  Project Overview

This project focuses on cleaning, preprocessing, analyzing, and visualizing the FIFA World Cup 2026 Player Performance dataset using Python. The project demonstrates essential data science techniques such as handling missing values, removing duplicates, detecting outliers, performing exploratory data analysis (EDA), and creating visualizations to extract meaningful insights.

##  Objectives

* To collect and analyze the FIFA World Cup 2026 Player Performance dataset.
* To clean and preprocess the dataset by handling missing values, duplicates, and outliers.
* To perform Exploratory Data Analysis (EDA) using Python.
* To create visualizations for understanding player and team performance.
* To identify patterns, trends, and insights from the dataset.

##  Dataset Information

* **Dataset Name:** FIFA World Cup 2026 Player Performance Dataset
* **File Format:** CSV
* **Domain:** Sports Analytics
* **Dataset Features:** Player statistics, goals, assists, ratings, team information, and performance metrics.

##  Technologies and Libraries Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Data Cleaning Process

The following preprocessing steps were performed:

* Imported the dataset.
* Checked the dataset structure and dimensions.
* Identified and handled missing values.
* Removed duplicate records.
* Detected and treated outliers.
* Corrected data types where required.
* Prepared the dataset for analysis and visualization.

##  Exploratory Data Analysis (EDA)

The dataset was analyzed to understand:

* Player performance distribution.
* Team-wise performance.
* Goal scoring patterns.
* Player rating analysis.
* Statistical summaries and correlations.

## Visualizations Created

The following visualizations were generated:

* Top Performing Players Bar Chart
* Team Performance Comparison Chart
* Player Rating Distribution Histogram
* Correlation Heatmap
* Goal Analysis Visualization

## Key Findings

* Compared overall team performance.
* Observed player rating distributions.
* Found relationships between different performance metrics.
